# Instacart MLOps Pipeline

End-to-end MLOps pipeline for product recommendation system using Instacart Market Basket Analysis dataset.

## Project Overview
Building a production-ready ML system with:
- Automated training pipelines
- Model monitoring and drift detection
- A/B testing framework
- Cost optimization strategies

## Tech Stack
- **ML**: Python, Scikit-learn, XGBoost
- **MLOps**: MLflow, DVC
- **CI/CD**: GitHub Actions
- **Deployment**: Docker, FastAPI
- **Monitoring**: Prometheus, Grafana
